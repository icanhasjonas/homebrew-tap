# Homebrew Tap

Homebrew formulae for [icanhasjonas](https://github.com/icanhasjonas) projects.

## Install

```bash
brew tap icanhasjonas/tap
brew install icloud-tools
```

## Formulae

| Formula | Description |
|---------|-------------|
| [icloud-tools](https://github.com/icanhasjonas/icloud-tools) | CLI for managing iCloud Drive files (replacement for brctl download/evict) |
